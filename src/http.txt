2. HTTP Methods 

1. GET : retrieve a message like data read from db 
2. HEAD : No message body { it just response headers only}
3. OPTIONS : what operation available on like /users pr post aur get aur kya kya 
4. TRACE: for debugging 
5. DELETE:delete a resource /remove
6. PUT: update a resource 
7. POST:mostly add /create
8. PATCH:change part of resource
